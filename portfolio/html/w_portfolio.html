ポートフォリオ


Yuri Yamauchi Portfolio
WEB DESIGN

概要

自身の経歴や制作物をまとめたポートフォリオサイトを制作しました。

目的

就職活動を行うにあたり、これまでの制作物やそれに対する思考、
経歴や強みを伝える。

ターゲット

採用担当者の方々や興味を持って下さった方々。

情報設計

最初に何ができるのかを伝える為、MVのすぐ下にWorksを配置し、
その下にAboutを配置しています。
トップページのWorks一覧は、一目見て何を制作したのかが伝わる
ように概要を記載。
またAboutでは自身の強みやこれまでの経歴を記載し、どんな人物かを
伝えています。

デザインプロセス

就職活動にあたり分かりやすくシンプルなデザインを心がけつつ、
自分に興味をもっていただけるよう自分らしさを表現しました。
フォントに関しては、タイトルにはシンプルで清潔感のある
「YuMincho」を使用し、本文にはデザインに馴染むように
「Zen Kaku Gothic New」を使用しています。

作成期間

企画･WF ２週間
デザイン 4週間

使用ツール

Figma ／ Visual Studio Code
